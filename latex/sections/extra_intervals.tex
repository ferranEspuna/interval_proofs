\section{Extra intervals beyond the equally spaced construction}
\label{sec:extra-intervals}

The first evidence for this phenomenon came from the general program
\code{circle_intervals.py}, in which all interval endpoints and lengths are
free.  For the small numbers of allowed intervals that were computationally
feasible, and for the tested multipliers \(m\leq10\), the general MILP again
reported the same structured optimum: an equally spaced base, together with
one short extra interval when an improvement was possible.  Thus this shape
arose without being imposed on the model.  These results concern the tested
finite-interval models and remain numerical rather than exact global upper
bounds.

The specialized script \code{equally_spaced_tiny_interval_lp.py} was written
afterward to test the suggested pattern for a much larger range of
multipliers.  It fixes the base parameters \(q,\delta,a\) from
Section~\ref{sec:equally-spaced-construction}, adds a chosen number of variable
intervals, and imposes the lifted constraints from
Section~\ref{sec:milp-formulation}.  Pairwise binary variables enforce
non-overlap because the extra intervals are not assigned to gaps in advance.
Unlike the general program, this is a local extension model: it asks whether
the fixed structured base can be enlarged, not whether that base is globally
optimal.  Its lighter formulation made it possible to test all multipliers up
to \(100\).  When it finds a positive extra length, the next step is to recover
rational endpoints and produce a finite exact certificate.

These small MILPs sometimes find that a short extra interval can be added to
the optimized equally spaced configuration.  The first example occurs for
\(m=4\), where the density improves from \(1/6\) to
\[
    \frac{11}{64}=\frac16+\frac1{192}.
\]
The exact three-interval certificate is given in
Section~\ref{sec:explicit-m4-m5}.

The specialized scan through \(m=100\) suggests that an improvement is
possible when
\[
    m\equiv0,1\pmod4,
\]
with \(m=5\) as an exception.  This is currently only a numerical pattern.
The desired theoretical output is an explicit formula for the location and
length of every added interval, followed by a finite certificate like those
in Section~\ref{sec:explicit-m4-m5}, and a proof that the equally spaced
construction is at least locally optimal when \(m\equiv2,3\pmod4\).

Concrete questions are:
\begin{enumerate}
    \item For which values of \(m\) can one add positive measure to the
          optimized equally spaced construction?
    \item Is one extra interval always enough to realize the best local
          improvement?
    \item Are the added intervals genuine new components, or do they arise
          from a different rounding of the same discrete construction?
    \item Does every improvement have size \(o(1)\), so that the asymptotic
          density remains \(1/8\)?
    \item More generally, is it true that
          \(\lim_{m\to\infty} d_m(\T)=1/8\)?
\end{enumerate}
